Google Doc Comment Collector Tutorial

Google Docs allows for comments to be added to documents. These comments are not straightforward to collate manually. The proposed project would develop a web application that would take in any number of Google Docs, extracts the comments automatically and store them into a database, the web application would allow the database to be queried by the user. The user can search a comment for a specific keyword of by date and author. Below is a vidieo giving a tutorial on how to use the applacation.
